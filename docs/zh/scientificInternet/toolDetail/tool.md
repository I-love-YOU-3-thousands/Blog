# 工具

::: tip 关于 VPN 与 代理工具
推荐使用列出的代理工具，不推荐使用任何 VPN 以及破解的 VPN

- VPN：点击连接即可使用，自带节点，但是会被国家审计，说错话可能要请去喝茶

- 代理工具：需要自备节点，导入后方可使用，部分提供节点的机场可以绕过审计

:::

::: warning 推荐使用：带 ⭐ 的有教程
工具很多，不必都会，选其一即可
:::
